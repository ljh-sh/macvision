## Add

- `macvision ocr <image>` — extract text with `VNRecognizeTextRequest`, language selection, confidence scores, bounding boxes.
- `macvision classify <image>` — scene/object labels with `VNClassifyImageRequest`; animal species with `--animals` (`VNRecognizeAnimalsRequest`).
- `macvision detect <image>` — faces, rectangles, barcodes/QR, text regions, and horizon in a single pass.
- `macvision document <image>` — document outline (quadrilateral) for crop/deskew via `VNDetectDocumentSegmentationRequest`.
- `macvision salient <image>` — saliency heatmap PNG (attention or objectness).
- `macvision feature <image>` — image feature-print vector; `--compare` for pairwise distance.
- `macvision daemon` — long-lived FIFO daemon serving NDJSON requests over named pipes.
- `macvision doctor` — environment and Vision capability report.
- Image input from a file path, `-` for stdin base64, `--clipboard`, or `--screen`.
- Compact JSON output for all commands; pixel (top-left origin) and normalized bounding boxes.

## Note

- Built on the in-repo `Cmd` framework (zero external Swift dependencies); `Vision` is a system framework, so there is no model to download.
- `Vision`'s `perform(_:)` is synchronous, so macvision does not start an `NSApplication` run loop for image work (unlike audio tools).
- Bounding boxes use top-left origin in pixels — the convention agents need for screen coordinates.
