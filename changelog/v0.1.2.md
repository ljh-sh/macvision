## Add

- `macvision --version` / `-V` prints the version (e.g. `macvision 0.1.2`).
- SLSA build provenance is now generated automatically on every release (baked into `release.yml`), instead of a manual `retrofit-slsa` step.

## Note

- This is the first release whose provenance is attached automatically by CI.
- Version is tracked in `Sources/util/Version.swift` (`MACVISION_VERSION`), bumped in the release PR for each tag.
