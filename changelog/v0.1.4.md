## Add

- **Language shortcuts** `--en`, `--zh`, `--ja`, `--ko` on `ocr` and `detect --ocr`. Each appends its script to `--lang` at the position it appears, so flag order is the language order:
  - `macvision ocr img.png --ja --en` == `macvision ocr img.png --lang ja-JP,en-US`
  - `--zh` adds both `zh-Hans` and `zh-Hant`.
- CJK-safe by construction: `--ja --zh` puts Japanese first, so kana isn't starved by an earlier Chinese entry (the Vision CJK-ordering gotcha documented in v0.1.3).

## Note

- A single shortcut **narrows** to that script (not the broad `all` default): `macvision ocr img.png --ja` → `[ja-JP]`. Combine freely with `--lang`, e.g. `--lang cjk --ja`.
- Under the hood: a generic `appendsTo` / `appendValue` on `OptMeta` — a boolean flag that appends a value to a `[String]` option in CLI order. No per-command special-casing.
- 4 new parser tests; all 19 tests pass.
